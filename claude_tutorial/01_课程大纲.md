# LightX2V 推理框架深度剖析课程大纲

## 课程目标
通过 15 节课深入理解 LightX2V 框架设计，最终能够：
- 掌握框架核心架构与设计模式
- 理解各优化技术的实现原理
- 具备快速适配新模型的能力
- 能够集成和开发新的优化手段

---

## 第一部分：架构设计（第1-5课）

### 第1课：整体架构与设计哲学
- 框架分层设计：Pipeline → Runner → Components
- 核心抽象：为什么这样设计？
- 注册机制：RUNNER_REGISTER 的实现与作用
- 配置驱动：JSON 配置如何映射到代码执行
- 扩展点设计：哪些地方预留了扩展能力

### 第2课：Pipeline 层设计
- LightX2VPipeline 的职责边界
- model_cls 到 Runner 的映射机制
- Offload 配置的传递与生效
- Generator 的创建与参数注入
- 多任务支持（T2V/I2V/T2I）的统一抽象

### 第3课：Runner 层设计
- BaseRunner → DefaultRunner → 具体Runner 的继承体系
- Runner 的核心职责：组件编排与生命周期管理
- 组件初始化顺序与依赖关系
- 推理循环的设计（encode → denoise loop → decode）
- WanRunner vs HunyuanVideoRunner 的差异分析

### 第4课：网络层设计（DiT）
- base_model.py 的抽象设计
- 三阶段推理：pre_infer / transformer_infer / post_infer
- 权重加载机制：weights/ 目录的组织方式
- infer/ 与 weights/ 的分离设计意图
- 如何支持不同的 DiT 变体

### 第5课：编码器与调度器设计
- 输入编码器的抽象：文本编码器、图像编码器
- 编码器的懒加载与 Offload 支持
- 调度器��口设计：scheduler.py
- Flow Matching vs DDPM 的调度器实现差异
- VAE 编解码的集成方式

---

## 第二部分：优化技术实现（第6-11课）

### 第6课：Offload 机制实现
- 显存压力的来源分析
- Phase 级 Offload：整个组件的迁移
- Block 级 Offload：DiT Block 粒度的迁移
- Disk Offload：磁盘缓存策略
- Offload 调度器的实现细节
- 如何为新模型添加 Offload 支持

### 第7课：注意力算子体系
- 注意力算子的统一接口设计
- Flash Attention 集成方式
- Sage Attention 集成方式
- 算子选择机制：attn_mode 参数的路由
- 如何集成新的注意力算子

### 第8课：稀疏注意力实现
- 稀疏注意力的理论基础
- SLA（Sliding Local Attention）实现分析
- SVG Attention 的稀疏模式设计
- Neighborhood Attention 实现
- sparse_mask_generator.py 的作用
- 稀疏模式的配置与调优

### 第9课：并行策略实现
- CFG 并行：正负样本的并行计算
- Ulysses 并行：序列维度切分
- Ring Attention：多卡注意力计算
- 分布式通信的封装
- 并行策略的组合使用

### 第10课：缓存加速实现
- 缓存加速的核心思想：跳过冗余计算
- TeaCache 实现原理与代码分析
- MagCache 实现原理与代码分析
- 缓存策略的配置体系
- 如何设计新的缓存策略

### 第11课：量化技术实现
- 量化的基本原理回顾
- FP8 量化的实现方式
- NVFP4 量化支持
- TorchAO 集成
- GGUF 格式支持
- 量化与其他优化的组合

---

## 第三部分：扩展与实战（第12-15课）

### 第12课：适配新模型（上）- 架构分析
- 新模型适配的整体流程
- 分析目标模型的架构特点
- 确定需要实现的组件
- Runner 的设计与实现
- 以 WorldPlay 为例的完整分析

### 第13课：适配新模型（下）- 代码实现
- 网络结构的实现（networks/）
- 权重加载的适配
- 调度器的适配或复用
- 配置文件的编写
- 测试与验证流程

### 第14课：服务化与分离部署
- Server 模块的架构设计
- API 层的设计（FastAPI）
- 任务管理与调度
- disagg 模块：分离部署的实现
- Encoder/Transformer/Decoder 服务分离
- Mooncake 集成原理

### 第15课：Kernel 开发与性能调优
- lightx2v_kernel 模块结构
- CUDA Kernel 的开发流程
- Triton 算子的使用
- 性能分析工具与方法
- 瓶颈定位与优化策略
- 端到端性能调优实战

## 每课结构

每节课将包含：
1. **设计分析** - 为什么这样设计，解决什么问题
2. **源码剖析** - 关键代码逐行分析
3. **类图/流程图** - 可视化理解
4. **扩展练习** - 动手修改或扩展
