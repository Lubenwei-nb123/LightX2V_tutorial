# LightX2V 推理框架学习路线图

## 框架概述

LightX2V 是一个轻量级视频生成推理框架，支持：
- 文本生成视频 (T2V)
- 图像生成视频 (I2V)
- 文本生成图片 (T2I)
- 图像编辑 (I2I)

核心优势：相比 Diffusers 等框架，单卡可达 1.9x 加速，8卡可达 3.9x 加速。

```

└── scripts/               # 运行脚本
```

## 推荐学习路线

### 第一阶段：快速上手（1-2天）

1. **阅读官方文档**
   - 中文文档：https://lightx2v-zhcn.readthedocs.io/zh-cn/latest/
   - 项目 README_zh.md

2. **运行示例代码**
   - 从 `examples/BeginnerGuide/ZH_CN/` 开始
   - 先跑通 `02.Wan21-1.3B.md`（小模型，资源需求低）

3. **理解基本用法**
   ```python
   from lightx2v import LightX2VPipeline

   pipe = LightX2VPipeline(
       model_path="...",
       model_cls="wan2.1",
       task="i2v"
   )
   pipe.enable_offload(...)
   pipe.create_generator(...)
   pipe.generate(...)
   ```

### 第二阶段：理解核心架构（3-5天）

1. **Pipeline 流程**
   - 入口：`lightx2v/pipeline.py` - LightX2VPipeline 类
   - 理解 `create_generator()` 和 `generate()` 的调用链

2. **Runner 机制**
   - 位置：`lightx2v/models/runners/`
   - 每个模型有对应的 Runner（如 WanRunner、HunyuanVideo15Runner）
   - Runner 负责协调编码器、DiT、VAE 的调用

3. **核心组件**
   - 输入编码器：`models/input_encoders/` - 处理文本/图像输入
   - 网络结构：`models/networks/` - DiT、VAE 等
   - 调度器：`models/schedulers/` - 扩散过程调度

### 第三阶段：深入优化技术（1-2周）

1. **显存优化**
   - Offload 机制：`lightx2v/common/offload/`
   - Block 级别 vs Phase 级别卸载

2. **注意力优化**
   - 配置文件：`configs/attentions/`
   - 支持 Flash Attention、Sage Attention 等

3. **量化技术**
   - FP8、NVFP4 量化
   - 查看 `configs/` 下的量化配置

4. **并行策略**
   - CFG 并行
   - Ulysses 并行

### 第四阶段：扩展与定制（持续）

1. **添加新模型支持**
   - 参考现有 Runner 实现
   - 理解注册机制：`RUNNER_REGISTER`

2. **自定义优化**
   - 算子层面：`lightx2v_kernel/`
   - 调度策略定制

## 学习建议

1. **边跑边学**：先跑通示例，再深入代码
2. **从小模型开始**：Wan2.1-1.3B 资源需求低，适合学习
3. **关注配置文件**：很多功能通过配置开启
4. **善用调试**：在关键位置加 print 理解数据流
